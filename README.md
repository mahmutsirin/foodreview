# Yemek Değerlendirme ve Yorumlama Platformu

Bu proje, kullanıcıların restoranları değil doğrudan yemekleri değerlendirebildiği ve puanlayabildiği bir web uygulamasıdır. Kullanıcılar belirli yemekler hakkında yorum yapabilir, puan verebilir, fotoğraf yükleyebilir ve diğer kullanıcıların içerikleriyle etkileşime geçebilir.

## Projenin Amacı

Geleneksel restoran değerlendirme sistemlerinden farklı olarak, kullanıcıların belirli yemekler hakkında deneyimlerini paylaşmalarını ve en iyi yemekleri keşfetmelerini sağlamaktır. Örneğin, kullanıcılar "En iyi makarna", "En iyi hamburger" veya "En iyi döner" gibi aramalar yaparak yemek bazlı sıralamalara ulaşabilirler.

## Özellikler

* Kullanıcı kayıt ve giriş sistemi
* Yemek ekleme ve listeleme
* Yemeklere puan verme
* Yemeklere yorum yapma
* Fotoğraf yükleme
* Yorum ve gönderi beğenme sistemi
* Yemek kategorilerine göre filtreleme
* Popüler ve yüksek puanlı yemekleri görüntüleme
* Kullanıcı profil sistemi
* Veritabanı destekli dinamik içerik yönetimi

## Kullanılan Teknolojiler

* HTML
* CSS
* JavaScript
* PHP
* MySQL
